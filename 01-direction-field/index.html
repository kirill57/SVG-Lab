Learning Objectives

  • Visualize the direction field for the differential equation \(y' = -y\).
  • Understand how an integral curve represents a specific solution based on an initial condition.
  • Manipulate the initial condition (\(y_0\)) by dragging an interactive handle on the y-axis.

Code Explanation

This module uses JavaScript to dynamically generate an SVG direction field. For each point on a grid, it calculates the slope \(m = -y\) and draws a small line segment. The red integral curve is the analytical solution to the equation, \(y(x) = y_0 e^{-x}\), which is recalculated whenever the user drags the blue handle to set a new initial condition \(y_0\). 

// The core logic for the solution curve
function updateIntegralCurve(y0) {
  // The solution is y(x) = y0 * e^(-x)
  let pathData = '';
  const steps = 200;
  // ... loop to generate path points ...
  for (let i = 0; i <= steps; i++) {
    const x = xRange.min + i * stepSize;

    // ...

  }
  integralCurve.setAttribute('d', pathData);
}
